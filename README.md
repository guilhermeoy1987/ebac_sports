# EBAC Sports - Integração Moderna com React-Redux

Este repositório foca na atualização da lógica de estado global da aplicação, utilizando os hooks mais recentes para conectar o React ao Redux Toolkit.

## O que foi realizado:
* **Refatoração com Hooks:** Implementação de `useSelector` e `useDispatch` para gerenciamento de dados.
* **Sincronização de Estado:** Garantia de que as ações do usuário (favoritos e carrinho) reflitam em tempo real em toda a interface.
* **Melhoria de Performance:** Otimização da forma como os componentes acessam a store global.
* **Código Limpo:** Redução de código boilerplate (repetitivo) através das práticas modernas do Redux Toolkit.

## Tecnologias:
* React.js
* Redux Toolkit (Hooks)
* JavaScript (ES6+)
* Git / GitHub (Branch: atualizadoreduxcomreact)
