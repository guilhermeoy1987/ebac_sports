🛒 EBAC Sports: State Management com Redux Toolkit
Refatoração de uma aplicação de e-commerce para gerenciamento de estado global e consumo de API otimizado.

✅ Redux Toolkit: Implementação de Slices e Store para gerenciar o estado global do carrinho.
✅ RTK Query: Substituição do fetch tradicional por hooks automáticos para consumo da API (useGetProdutosQuery).
✅ Hooks Avançados: Uso de useSelector para leitura de dados e useDispatch para disparo de ações.
✅ Modern React: Componentização limpa com remoção de prop drilling e estados locais (useState).

Stack: React, Redux Toolkit, RTK Query, TypeScript e Styled Components.
