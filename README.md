# EBAC Sports - E-commerce com React e Redux

Este repositório contém uma aplicação de loja de artigos esportivos desenvolvida para praticar o gerenciamento de estados complexos e integração com APIs.

## O que foi realizado:
* **Estado Global com Redux:** Configuração de uma store central para gerenciar carrinho e favoritos.
* **Lógica de Compra:** Implementação de funções para adicionar e remover produtos do carrinho.
* **Sistema de Favoritos:** Criação de funcionalidade para marcar e desmarcar itens de interesse.
* **Consumo de API:** Busca dinâmica de produtos para exibição na vitrine da loja.

## Tecnologias:
* React.js
* Redux Toolkit
* JavaScript (ES6+)
* Styled Components (ou CSS tradicional)
* Git / GitHub
